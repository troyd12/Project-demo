# TD Movement — Troy Dixon Portfolio

Relentless Advancement. Relentless Breakthroughs. Relentless Movement.

## About

Personal portfolio site for Troy Dixon — Lead Technical Recruiter specializing in engineering talent acquisition, AI automation, and bridging the gap between engineering teams and the talent they need.

## Tech Stack

- **HTML/CSS/JS** — Single-file architecture
- **Fonts** — Bebas Neue + Space Grotesk (Google Fonts)
- **Hosting** — Netlify
- **Domain** — [tdmovement.com](https://tdmovement.com)

## Features

- Animated intro sequence with per-letter fly-in
- Neural network constellation background (canvas)
- Custom volt green cursor with lerp tracking
- Scroll-reveal animations via IntersectionObserver
- Project modals with detailed case studies
- Skills filter with category tabs
- Live stats counter
- Fully responsive (mobile-first breakpoints)
- Dark/light theme toggle

## Local Development

Open `index.html` directly in a browser — no build step required.

## Deployment

Push to `main` branch. Netlify auto-deploys from this repo.
